## 运行

  安装相关依赖
  ~~~bash
  pip install -r requirement.txt
  ~~~
  直接运行app.py
  ~~~bash
  python app.py
  ~~~
  然后在浏览器登录
  ~~~
  127.0.0.1:8080
  ~~~
